The following program utilizes AWS text-to-speech service that uses advanced deep learning technologies to synthesize speech that sounds like a human voice. 

The code I have written is just a basic program that downloads the text from a broadway script, acquire all the different character's names and output mp3 files with designated voices of each of the characters' lines.

Dependencies: 
In order to make this work, you need to install AWS onto your computer and follow the directions using the following link: https://docs.aws.amazon.com/polly/latest/dg/setup-aws-cli.html


Next, just go into your Python IDE, and run the code. the output should be each slide_#.mp3 file in the given language.
