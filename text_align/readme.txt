The following program compares two fav files of the same play and matches up the timing based on the timing of the old play. 


Directions: 

Place the two files into the same directory as the py files. 

Type the following into the command line: 

python new_fav_converter.py old_show_fav.txt new_show_fav.txt

there will be an output called new_show_fav.txt and other files like compare and compare2 texts files that help for debugging in certain cases.
